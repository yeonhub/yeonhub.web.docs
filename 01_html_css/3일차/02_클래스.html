        클래스 선택자 : 별명/별칭 (10점)
                        한 페이지에 여러번 사용 가능
                        형식)
                        <태그 class="클래스명"
                        style: .클래스명
                        
                        클래스명 규칙
                        1. 영문자, 숫자, _, -
                        2. 첫 글자는 숫자 제외, 주로 영문자 소문자 사용
                        3. 대소문자 구별함
                        4. 특수문자, 한글 사용 불가
                        5. 두 단어가 합쳐질 경우
                            1) 단어-단어 : dog-cat
                            2) 단어_단어 : dog_cat
                            3) 단어단어(첫대문자) : dogCat
                        6. 주로 영역(div 집) 클래스를 작성하는게 좋다
