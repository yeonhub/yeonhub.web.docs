        선택자 {속성:값; 속성:값; }
        선택자 {속성:값: 속성:값 } 
        맨 끝 속성은 ; 생략 가능
        태그, 아이디, 클래스 등

        color : 글자색
        background-color : 배경색
        font-size : xxpx : 글자 크기
        width : 너비 px
        height : 높이 px

        우선순위
        *(모든) : 0점
        태그 : 1점
        .(클래스) : 10점
        :(가상선택자) : 10점
        #(아이디) : 100점

        연습
        #wrap : 100
        #wrap #header : 100+100=200
        .gnb li : 10+1
        .gnb li a : 10+1+1
        ul li a : 1+1+1
        a:hover : 1+10
