1. hn : h1~h6 - 제목
        2. p - 단락, 문단
        3. br - 강제 개행(빈태그-열고 닫는게 없음 혼자 쓰임)
        4. ol - 순서 있는 목록 
                자식 li만 가능, li는 모든 태그 사용 가능
        5. ul - 순서 없는 목록 (비슷한 형태, 성격 나열)
                자식 li만 가능, li는 모든 태그 사용 가능
                예) 배너, 갤러리, 메뉴
        6. dl - 용어 정의
                자식 dt, dd만 사용 가능
                dt(용어 설명) 안에는 인라인 형식만
                dd(용어 설명) 모든 태그 사용 가능
        7. img - 이미지
                속성
                src : 이미지 주소
                alt : 대체 텍스트
                width : 너비
                height : 높이
                title : 도움말
        8. a - 링크
                속성
                href : url 주소
                target : _blank (새 탭)
                title : 도움말
        9. div - 공간 분할 (의미 없음, 남용 X )
                css 화면 결과 구조 
                그룹핑
        10. strong - 강강조
            em - 약강조
            span - 의미 없는 영역(작은)-(큰 영역은 div)
            b - 의미 없는 영역을 잡을때
            i - 폰트 아이콘 표현
