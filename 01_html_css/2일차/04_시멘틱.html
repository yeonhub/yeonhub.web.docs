저작권 패밀리 사이트 정책 
        ///////////////////////////
        
        1. header : 사이트 상단 머리 부분
                    로고, 메뉴, 상단메뉴(로그인, 회원가입 등)
                    전체메뉴(햄버거메뉴), 장바구니, 검색

        2. nav : 중요 메뉴

        3. footer : 하단, 하단 로고, 패밀리 사이트, 정책, 저작권

        4. section : 주제가 동일한 페이지 또는 컨텐츠 영역
                    반드시 h1~h6 제목이 있어야 한다

        5. article : 독립된 작은 컨텐츠

        6. aside : 컨텐츠 외에 나머지 영역(하단)

        7. main : 실제 컨텐츠 영역(본문)
