        transform : 
        위치 : 자기 자신 기준
        translate(x, y)
        translateX(x)
        trasnlateY(y)

        크기 : 2=2배, 3=3배
        자식들도 같이 크기 적용
        scale(너비, 높이)
        scaleX(너비)
        scaleY(높이)

        회전 : 
        rotate : 360도 기준 deg 양수, 음수 가능

        양수            음수
                 (0)
        (-90)           (90)
                (180)
        
        rotateX() : x축 기준 회전
        rotateY() : y축 기준 회전

        기울기 : 양수, 음수 가능 
        skew(x, y)
        skewX(xdeg)
        skewY(ydeg)
