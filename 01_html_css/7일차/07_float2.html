        박스 옆으로 나열
        1. display: inline-bolck(거의 사용X)
        2. float : 띄워서 나열 (15% 사용)
        나머지 대상들도 옆으로 따라서 나열
        left 띄워서 왼쪽부터 나열
        right 띄워서 오른쪽부터 나열
        none 적용 안됨

        clear
        left : 왼쪽 해제
        right : 오른쪽 해제
        both : 둘 다 해제

        * 자식을 띄우면 부모가 자식을 인식하지 못한다 

        해결 
        1. 부모에 overflow:hidden 
        2. 부모에 ::after { content:""; display:block; clear:both } 
          주로 .clearfix::after { content:""; display:block; clear:both } 만들고 사용한다.

        float 알아야 하는 이유
        1. 유지/보수
        2. flex가 안될때 해결
