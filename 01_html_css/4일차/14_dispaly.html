강강조 작은 영역 
        ------------------
        display : 화면에 보이는 성격 변경(실제값 변경X)
        inline : 블럭 성격을 인라인으로 변경(거의 사용X)
        block : 인라인 성격을 블럭으로 변경(많이 사용)
                화면에 보이기(js)
                none : 화면에서 사라지기(js)
        inline-block : 작은 영역을 박스 옆으로 나열(상단 유틸 메뉴)
                    주로 flex 사용
