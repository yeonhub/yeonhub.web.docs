        text-decoration
        https://developer.mozilla.org/en-US/docs/Web/CSS/text-decoration

        text-decoration: underline; : 밑줄
        디자인으로 사용하지 않음
        text-decoration: none; *

        접근성 있는 사이트 포털(네이버/네이트/다음...)
        마우스 올렸을 때 : hover (밑줄)

        접근성 없는 사이트 : X

        링크 : a (파란 밑줄 제거 할때)
        text-decoration : none;
        a는 컬러가 상속되지 않음, a 컬러 변경
