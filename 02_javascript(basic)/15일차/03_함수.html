함수 : 자주 사용하는 명령(실행문, 내용, 문장, 수식, 제어문등)들을 담아두었다가 필요할때 꺼내 사용하는 형태
        장점 : 코드 줄이기, 코드 재사용

        ====================== 형식-1 ======================
        function 함수명() { //함수명규칙=변수규칙동일
            실행문
        } - 정의문
        함수명() : 호출

        ====================== 형식-2 ======================
        function 함수명(매개변수, 매개변수) {
            실행문
        }
        함수명(값, 값)

        ====================== 형식-3 ======================
        function 함수명(매개변수, 매개변수) {
            실행문
            return 반환값
        }
        let 변수명 = 함수명(값, 값)
